# Learning_Python_Day10

Today's topic is dictionaries!

**Goal:** Master dictionaries to store and retrieve key-value pairs for real-world data

- **task1:** Create a `person` dictionary with keys: name, age, city

- **task2:** Add new key-value pairs to the existing dictionary

- **task3:** Update values using both `dict[key] = value` and `.update()`

- **task4:** Use `.get()` with a default value if the key does not exist

- **task5:** Loop through the dictionary using `.items()`

- **task6:** Create a nested dictionary (dictionary inside a dictionary)

**mini-project:**
Student Database Manager:  
Create a program that stores multiple students in a nested dictionary.

- Each student should have: name, age, grade, and city  
- Allow adding new students  
- Allow updating student information  
- Display a single student using `.get()`  
- Loop through all students and print their details clearly
